Data Analysis

This page looks at the downsampled normalized datasets, and the autocorrelation (ACF) of these.

Normalized Data Comparison

The normalized data can be plotted in the same graph. Visual inspection reveals a clear correlation between tweet sentiment radicalness and number of tweets posted, although there is no immediately obvious correlation with the GBP EUR exchange rate. It is clear there is periodicity in the Number of Tweets and Sentiment radicalness.

AutoCorrelation for each normalized downsampled Dataset

The autoCorrelation (ACF) was implemented. It was found there was a clear correlation for the average hourly number of tweets and tweet sentiment radicalness, per 24 hours. However, the average tweet sentiment and exchange rates were clearly not autocorrelated.

Scatter Matrix

The scatter matrix below allows us to quickly visually inspect any correlations between any of the datasets. While some of the attributes of the tweets are slightly correlated, there is little to no linear correlation between exchange rate or any other variable.

The correlation of Tweet polarity to Exchange rate = 0.076

The correlation of Tweet polarity to Number of tweets = 0.10

The correlation of Tweet polarity to Tweet Radicalness = 0.63

The correlation of Exchange rate to Number of tweets = 0.072

The correlation of Exchange rate to Tweet Radicalness = 0.1

The correlation of Tweet Radicalness to Number of tweets = 0.68
