Insights

Innovation Opportunities

• This data could, with a live working platform and a larger backend storage solution, help to either calculate the populations' mood, or the exchange rate based off of tweet sentiments.

• This dataset would grow larger and larger, with the need for more complex data analyses and machine learning software.

• The twitter sentiment need not only apply to GBP EUR exchange rate (as Brexit will eventually pass), but could also help predict other stock prices, depending on certain keywords.

Creativity Opportunities

• The twitter sentiment need not only apply to GBP EUR exchange rate (as Brexit will eventually pass), but could also help predict other stock prices, depending on certain keywords.

• The data can also help infer other meaning. For example, there might be a spike in twitter activity and a drop in the pound value if there is a negative news article released on the topic, and tools could be implemented to understand how these articles may affect future GBP EUR values.

Enterprise Opportunities

• The data could be used to work out the best times to travel abroad, for example to take advantage of the best exchange rates. Alternatively it could warn you when not to.

Scalability Opportunities

• Growing the machine learning model such that it can predict future values would be beneficial to all. Currently only present values are predicted, and with much more data and more in-depth analysis, the database could become rather robust.

Complexity

• Adding more and more data sets other than just twitter, which may be more correlated with the exchange rate. For example, other exchange rater such as USD, or stock price shares of british companies. All these factors could add up to a more robust model.
