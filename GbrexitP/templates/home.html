GbrexitP

Dataset 1: The GBP - EUR Exchange Rate

This data was collected from the AlphaVantage API every 3 minutes between 28th December 2018 and 7th January 2019.

Dataset 2: Tweets containing the word 'Brexit'

Tweets containing the word 'Brexit' were collected from the Twitter Developer API every 3 minutes from 28th December 2018 to 7th January 2019.

Aims

This website aims to address 4 key areas, which can be accessed along the navigation bar:

• Raw Data Visualisation: plotting the data sets, allowing you to visually inspect any relationships between the data

• Data Analysis: Normalizing, downsampling, smoothing, and autocorrelating the data, as well as correlating the data amongst themselves. This analysis can help choose variables for a machine learning model.

• Predictions: Using a machine learning model based off of tweet data, a rough estimate for the exchange rate can be calculated.

• Insights: How can these conclusions and datasets be scaled, used for creative enterprise opportunities and further analysed for future improvements.
