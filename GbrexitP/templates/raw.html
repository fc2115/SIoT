Individual Tweet Sentiment over Time

The tweet 'sentiment' is determined through a machine learning module called 'textblob', with > 0 positive, == 0 neutral, and < 0 negative.

The data is noisy due to the large number of tweet sentiments analysed. The yellow line shows the downsampled average tweet sentiment every 3 minutes. It can be seen that generally, sentiments are less radical between midnight and 7am UTC. Finally, it can also be seen that tweet sentiment becomes more radical when more tweets are posted.

Tweet Sentiment Radicalness over Time

This graph measures the radicalness of tweet sentiments over time. Firstly, a peak analysis of the data initially found the symmetrical pattern about the x axis, which reflects how radical the tweet sentiments are. Next, the data is downsampled via the sum method, such that the values of the peaks is plotted. This clearly shows an increase in tweet sentiment radicalness during the day, and a clear reduction overnight (backing up the previous statement). It can also be seen on new years eve there was a spike in sentiment radicalness, which is likely positive.
